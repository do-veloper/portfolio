몸가짐 APP DESIGN

STYLE GUIDE

NOTIFICATION

가상의 헬스, 필라테스, 태닝샵 등 검색 및 정보
검색 APP을 창조한 작업물 입니다.

FONT

MAIN-FONT : 에스코어드림

SUB-FONT : IBM Plex Sans

깔끔하고 심플한 느낌을 주기 위해 에스코어드림 폰트를 채택하였으며
부분적으로 구분감을 주기 위해 IBM Plex Sans 폰트를
서브폰트로 사용하여 제작하였습니다.

COLOR

#F90150

#EEEEEE

#FFFFFF

차분하고 심플한 느낌을 주기 위해 전체적으로
최소한의 컬러만 사용하였으며 채도가 낮은 메인 색상과
무채색만 이용하여 디자인 하였습니다.

LOGO-DESIGN

몸가짐의 'M'을 변형. 위에서 바라본 어깨운동을 하는 사람을 모티브로
앱 로고를 제작했으며, 로고의 덤벨(아령)은 열정의 의미를 담아
빨간색으로 디자인 하였습니다.
APP LOADING

앱 실행시 처음 나타나는 로딩 화면으로
운동을 열심히 하고 있는
캐릭터 일러스트를 넣어 제작하였습니다.

LOG-IN

앱 접근성을 위해
네이버 로그인, 페이스북 로그인 할 수 있습니다.

해당 로그인 페이지에서 ID 찾기
비밀번호 찾기, 회원가입까지 가능하고

자동 로그인 버튼을 터치하면
다음 앱 접속시 자동 로그인

아이디 저장 버튼을 터치하면
비밀번호만 입력하면 로그인 할 수 있습니다.

MAIN

로그인 후 첫번째로 마주하는
메인 화면 입니다.

메인 슬라이드를 스와이프 하면
다른 이벤트 배너를 확인할 수 있고
해당 배너를 터치하면
관련 페이지로 이동합니다.

아래 각 카테고리를 터치하면
내 주변 센터 및 샵들을 추천해줍니다.

SEARCH

최하단 두번째 검색 아이콘을 터치하면
해당 화면으로 이동합니다.

검색창에 매장 등을 검색할 수 있으며
바로 아래에 최근 검색어를 나타냅니다.

아래에 근처 센터 및 샵 중
리뷰가 많거나 별점이 높은 가게를 추천합니다.

뜨고 있는 매장은
별점이 높거나, APP과 제휴를 맺은 매장을
랜덤으로 보여지게 합니다.

LIST-UP

내 주변 및 지역을 선택,
관심 운동을 고를 수 있는 필터 기능입니다.

혜택 매장을 터치하면 이벤트 진행중인 매장을
필터링하여 나타냅니다.

필터기능 및 위치를 확인할 수 있는
아이콘이 존재하며 터치하면 작동합니다.

리스팅 된 매장을 터치하면
DETAIL 페이지로 이동합니다.

DETAIL

매장을 터치하면 나오는
매장 DETAIL 페이지 입니다.

시설 둘러보기, 매장 위치, 전화하기
버튼이 있으며 터치하면 각 기능이 동작합니다.

사진을 스와이프하면 매장 사진을
여러장 볼 수 있습니다.

아래로 스와이프하면 회원권 등의 정보가 가격과 함께
보여지며 터치하면 구매 패이지로 이동합니다.

VIEW HISTORY

해당 페이지에서
최근 내가 봤던 매장들을 리스팅 해둔 페이지로

리스팅 된 목록들을 터치하면
매장 DETAIL 페이지로 이동합니다.

해당 페이지는 최하단 3번째 아이콘을
터치하면 바로 들어올 수 있습니다.

MY PAGE

최하단 4번째 아이콘을 터치하면
해당 마이페이지로 진입합니다.

쿠폰, 구매내역, 찜목록 등을 확인 할 수 있고
APP 정보, 고객센터 등 터치하면 해당 페이지로
이동합니다.
