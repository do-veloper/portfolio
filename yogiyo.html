STYLE GUIDE

FONT

MAIN-FONT : 여기어때 잘난체

SUB-FONT : 엘리스디지털배움체 R

요기요의 이미지에 맞는 동글동글하고 귀여운 폰트를 메인으로,
반응형으로 제작하여 창을 줄였을때 폰트가 눈에 잘 안들어오는 점을
고려해 엘리스디지털배움체를 sub-font로 채택하였습니다.

COLOR

#F90150

#F4C221

#6200B3

요기요의 대표 컬러인 F90150 색상과, 이전 로고 컬러를 참고해
F4C221 컬러를 더했으며, 웹 이용자에게 알록달록하고 친숙한 이미지로
기억되고 싶은 점을 의도하여 6200B3 컬러도 함께 추가하여 제작하였습니다.

LOGO-DESIGN

요기요의 'ㅇ/ㄱ/ㅇ' 자음을 따서 눈 코 입을 만들고
배달 메뉴를 고르는 사용자의 설레는 마음을 이미지화 하여
로고로 제작하였습니다.
STEP.1

메인 모델을 '맛있는 녀석들'의 출연진 개그맨 문세윤님으로 선정,
nav, mypage는 클릭 이벤트로 제작 하였습니다.

첫번째 화면에서 button 및 배너를 배치하였고,
해당 영역 클릭시 해당하는 페이지로 이동할 수 있게 작업 하였습니다.

사용성 테스트 결과 여기어때 잘난체를 사용했을때
가독성이 떨어지는 부분에 대한 수정을 하였으며
오른쪽 앱다운로드 quick-menu의 닫기 버튼이 필요하다는
의견을 수렴해 X버튼 클릭 이벤트를 추가 하였습니다.

STEP.2

YOGIYO APP의 편리함을 착안해 APP을 많이 참고하여
디자인 및 퍼블리싱 하였습니다.

STEP 1,2에 입력 후 검색을 진행하면 편리하게 주문할 수 있게
작업 하였습니다.

하단의 음식 종류별 필터, 프렌차이즈 음식 주문도 가능하게끔
작업 진행하였고 편리함에 중점을 두어 제작하였습니다.

STEP.3

공지사항/이벤트/제휴카드는 탭메뉴로 구성하여
해당 메뉴를 클릭하면 정보를 확인할 수 있게 작업하였습니다.

APP에서의 좋은 기능 중 하나라고 생각했던 실시간 인기검색어를
웹으로도 가져와 작업하였고,
요기요의 강점인 편의점 배달 및 주변 편의점 검색 또한 웹으로 도입
해당 기능을 이용할 수 있게끔 작업 진행 하였습니다.

다양하고 많은 업체와의 협업이 중요한 요기요,
해당 부분의 중요성을 높여 입점 신청하기 항목을 배치 하였으며
footer 위 요기요 SNS로 연결하는 아이콘까지 배치 하였습니다.
