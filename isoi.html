STYLE GUIDE

FONT

SUB-FONT : 여기어때 잘난체

아이소이의 바른 이미지를 살려줌과 동시에 가독성이 높은 폰트와
포인트를 줄 수 있는 폰트인 잘난체를 선정하였습니다.

COLOR

#448317

아이소이의 착한 성분, 자연친화적인 브랜드 이미지를 살리기 위해
자연적인 컬러와, 채도가 낮은 파스텔톤의 색감을 배치하려고
노력하였습니다. 제품 컷에는 해당 제품의 컬러코드를 활용해
채도를 살짝 높여 배경색으로 선정하였습니다.
STEP.1

메인 모델을 아이소이 브랜드 이미지와 잘 어울리고
실제 아이소이 전 모델 가수겸 배우 '아이유'님으로 선정하여
메인 슬라이드 첫번째 이미지로 배치하였으며
햄버거 메뉴 클릭시 NAV가 노출되며, 버튼은 X버튼으로 변경됩니다.

슬라이드는 총 3페이지로, 좌우 버튼을 만들어
사용자 편의에 따라 좌우버튼 클릭시 이동하게 작업 하였습니다.

아래는 배너 항목으로, 해당 배너 클릭시 해당 제품 소개 및
구매할 수 있는 페이지로 이동 할 수 있게 A링크로 작업하였습니다.

STEP.2

아이소이 대표 LINE UP별 제품 정보를 볼 수 있는 항목으로
해당 배너 클릭시 해당 LINE UP 제품 구매 페이지로
이동합니다. 해당 제품과 잘 매치되는 배경 컬러로 작업했습니다.

STEP.3

아이소이 전 모델 배우 '천우희' 님을 2번째 모델로 선정하여
작업하였으며, 해당 재생버튼 클릭시 아이소이 광고 영상을 재생할 수 있게
작업하였습니다.

하단에는 판매 상품을 리스트업하여 4*2로 배치하였으며
해당 제품 클릭시 구매할 수 있는 페이지로 이동할 수 있게
작업 하였습니다.
